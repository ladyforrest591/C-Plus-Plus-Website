C++ Basics

This section covers the fundamental concepts of C++ programming.

Printing

Basic Printing of a string in C++:

#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}

More Printing:

There are multiple ways to print strings in C++. Displayed are some popular and easy ways:

  • Basic Printing with `std::cout`
  • Printing with variables
  • `Printf` using `cstdio`
Basic Printing:
#include <iostream>

    int main() {
    // Basic Printing with std::cout
    std::cout << "Hello, World!";
    std::cout << "Hello, World!" << std::endl; // std::endl; makes a new line
    return 0;
}
Printing with Variables:
#include <iostream>

    int main() {
    // Printing using one variable
    int number = 0;
    std::cout << "The integer value is " << number << std::endl;

    // Printing using two variables or more
    int numberA = 10;
    int numberB = 20;
    int numberC = 30;

    std::cout << "Number A's value is: " << numberA << "Number B's value is: " << numberB << std::endl;
    std::cout << "Number A's value is: " << numberA << "Number B's value is: " << numberB << "Number C's value is: " << numberC << std::endl;

    return 0;
}
Basic Printing with Printf:
#include <cstdio>

    int main() {
    // Printing normally
    printf("Hello, World!\n"); // Similar to Python's syntax

    // Printing one variable
    int number1 = 10;
    printf("The number I am thinking of is: %d\n", number1);

    // Printing two variables
    int number2 = 20;
    printf("The numbers I am thinking of are: %d and %d\n", number1, number2);
    return 0;
}

More about Printing With Printf

Printing with Printf requires very specific syntax. Here are some common format specifiers:

  • `%d` for integers
  • `%f` for floating-point numbers
  • `%s` for strings
  • `%c` for characters
  • `%lf` for double
  • `%u` for unsigned int
  • `%x` for hexadecimal format
  • `%o` for octal format
  • `%%` to print a percent sign
Variables with Printf:
#include <iostream>

    int main() {

    // Printing an integer
    int age = 30;
    printf("My age is: %d\n", age);

    // Printing a double (more precise float)
    double pi = 3.14159;
    printf("The value of PI is: %lf\n", pi); // %lf for double

    // Printing a float (with exact precision)
    float temperature = 25.678f;
    printf("Current temperature: %.2f degrees Celsius\n", temperature); // .2f for 2 decimal places

    // Printing a character
    char grade = 'A';
    printf("Your grade is: %c\n", grade);

    // Printing a string
    const char* name = "Alice";
    printf("My name is: %s\n", name);

    // Printing multiple variables in one statement
    int item_count = 5;
    double price_per_item = 10.99;
    printf("You bought %d items at $%.2lf each.\n", item_count, price_per_item);

    // Using width specifiers and flags
    int value = 123;
    printf("Right-justified with width 5: %5d\n", value);
    printf("Left-justified with width 5: %-5d\n", value);
    printf("Zero-padded with width 5: %05d\n", value);

    // Printing hexadecimal and octal numbers
    int hex_num = 255;
    printf("Decimal: %d, Hexadecimal: %x, Octal: %o\n", hex_num, hex_num, hex_num);

    return 0;
}

Variables

Declaring Types of Variables:

There are many different kinds of variables in C++. Here are some examples:

  • `int`: Integers
  • `float`: for floating-point numbers
  • `double`: for double-precision floating-point numbers

Arithmetic Operators:

Arithmetic operators are used to perform mathematical operations. Here are some common arithmetic operators:

1. Addition (+) - Adds two operands

2. Subtraction (-) - Subtracts two operands

3. Multiplication (*) - Multiplies two operands

4. Division (/) - Divides two operands

5. Modulus (%) - Returns the remainder of division of two operands

#include <iostream>

    int main() {
      int a = 1;
      int b = 2;

      std::cout << "Addition: " << (a + b) << std::endl;
      std::cout << "Subtraction: " << (a - b) << std::endl;
      std::cout << "Multiplication: " << (a * b) << std::endl;
      std::cout << "Division: " << (a / b) << std::endl;
      std::cout << "Modulus: " << (a % b) << std::endl;

      // Example of using multiple arithmetic operators:
      int c = 3;
      int d = 4;
      int e = 5;
      int f = 6;
      int result = ((a + b - c) * (d / e) * f) % a
      std::cout << "The result of ((1+2-3)*(4/5)) % 1 is: " << result << std::endl;
    
    return 0;
}

Relational Operators:

Relational operators are used to compare two values. Here are some common relational operators:

1. Equal to (==) - Checks if two operands are equal

2. Not equal to (!=) - Checks if two operands are not equal

3. Greater than (>) - Checks if the left operand is greater than the right operand

4. Less than (<<) - Checks if the left operand is less than the right operand

5. Greater than or equal to (>=) - Checks if the left operand is greater than or equal to the right operand

6. Less than or equal to (<<=) - Checks if the left operand is less than or equal to the right operand

#include <iostream>

    int main() {
      int a = 5;
      int b = 10;

      std::cout << "a == b: " << (a == b) << std::endl; // false
      std::cout << "a != b: " << (a != b) << std::endl; // true
      std::cout << "a > b: " << (a > b) << std::endl;   // false
      std::cout << "a < b: " << (a < b) << std::endl;   // true
      std::cout << "a >= b: " << (a >= b) << std::endl; // false
      std::cout << "a <= b: " << (a <= b) << std::endl; // true
      
    return 0;
    }

String Methods:

In C++, the std::string comes with it many functions. Some include:

1. length() and size() - These return the number of characters in the string

2. substr() - This returns a substring from the string

3. append() - This adds characters to the end of the string

4. find() - This finds a substring within the string and returns its position

5. replace() - This replaces a portion of the string with another string

6. at() - This returns the character at a specific index

7. clear() - This clears the contents of the string

8. empty() - This checks if the string is empty

9. erase() - This erases characters from specified position and length


#include <iostream>
#include <string> // Required for std::string

int main() {
    // Using length() and size()
    std::string myString = "Hello, World!";
    std::cout << "Length of string: " << myString.length() << std::endl;
    std::cout << "Size of string: " << myString.size() << std::endl;

    // Using substr()
    std::string sub = myString.substr(7, 5); // "World"
    std::cout << "Substring: " << sub << std::endl;

    // Using append()
    myString.append(" How are you?");
    std::cout << "Appended string: " << myString << std::endl;

    // Using find()
    size_t pos = myString.find("World");
    if (pos != std::string::npos) {
        std::cout << "'World' found at position: " << pos << std::endl;
    } else {
        std::cout << "'World' not found" << std::endl;
    }

    // Using replace()
    myString.replace(7, 5, "C++");
    std::cout << "Replaced string: " << myString << std::endl;

    // Using at()
    char ch = myString.at(1);
    std::cout << "Character at index 1: " << ch << std::endl;

    // Using clear()
    myString.clear();
    std::cout << "Cleared string: '" << myString << "'" << std::endl;

    // Using empty()
    if (myString.empty()) {
        std::cout << "String is empty" << std::endl;
    } else {
        std::cout << "String is not empty" << std::endl;
    }

    // Using erase()
    std::string anotherString = "Hello, C++!";
    anotherString.erase(5, 4); // Erase ", C"
    std::cout << "Erased string: " << anotherString << std::endl;

    return 0;
}
